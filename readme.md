# 关于本项目

这个项目是从[vue3-grid-layout](https://github.com/xhlife/vue3-grid-layout)分支过来，它本身是基于[vue-grid-layout](https://github.com/jbaysolutions/vue-grid-layout)、实现对vue3支持。本项目拟实现以下特性：

- 在多个网格间移动GridItem
- GridItem高度自适应
- 暴露一些属性以定制外观
